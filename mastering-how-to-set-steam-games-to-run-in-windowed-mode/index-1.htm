Mastering How to Set Steam Games to Run in Windowed Mode

  • Post author:
  • Post published:December 26, 2025
  • Post category:Blog

Playing games on Steam requires ensuring compatibility with your computer’s hardware and software environment. When games are not optimized for your system’s CPU, graphics card, audio and video drivers, or internet connection, you may encounter errors or experience poor performance. Knowing how to switch Steam games between full-screen and windowed modes can significantly improve your gaming experience, especially if you face issues like screen freezing or crashes. This comprehensive guide will walk you through various methods to open Steam games in windowed mode, helping you troubleshoot performance problems and customize your gameplay.

—

How to Launch Steam Games in Windowed Mode?

Running Steam games in windowed mode can resolve low-performance issues during gameplay. While switching to full-screen mode is straightforward, enabling windowed mode often involves more steps. Adjusting launch options can also help fix internal server issues and optimize game performance. Let’s explore the different techniques to achieve this.

—

Method 1: Use In-Game Settings

The simplest way to run a game in windowed mode is through its internal settings. Most games provide display options where you can select the preferred mode. This approach avoids the need to modify launch parameters.

1. Launch the game via Steam and navigate to its video or display settings.

2. Look for the Display Mode or similar option, which is typically set to Full-screen by default.

3. Open the dropdown menu and select Windowed Mode or a comparable option.

4. Save your changes by clicking Apply or Save.

5. Exit Steam and relaunch the game to verify it opens in windowed mode.

This method is effective if the game’s settings permit mode switching. If not, proceed to alternative methods.

—

Method 2: Use Keyboard Shortcuts

If in-game settings do not allow changing the display mode, keyboard shortcuts can often help:

1. Launch the game normally.

2. While the game is running, press Alt + Enter simultaneously.

3. The display should toggle between full-screen and windowed mode instantly.

This quick shortcut works with many games and is useful for immediate adjustments. For more persistent changes, you may need to modify launch parameters.

—

Method 3: Change Steam Launch Options

Adjusting launch options in Steam allows you to set games to start in windowed mode automatically every time you launch them. This is useful for troubleshooting or if you prefer to always run in windowed mode.

1. Open Steam and go to your Library.

2. Right-click on the game you want to modify and select Properties.

3. In the General tab, click Set Launch Options….

4. In the input box, type `–windowed` to force windowed mode.

5. Click OK to save changes.

6. Close the properties window and launch the game to confirm it opens in windowed mode.

To further customize the window size, you can add parameters like `–windowed -w 1024 -h 768`, which sets the width and height explicitly. For more advanced configurations, refer to the game’s documentation or community forums. To verify game integrity after making changes, you can use Steam’s built-in tool here.

—

Method 4: Modify Game Launch Parameters via Shortcut

Another effective method involves editing the game shortcut directly:

1. Locate the game shortcut on your desktop.

2. Right-click and select Properties.

3. Switch to the Shortcut tab.

4. In the Target field, append `–windowed` at the end of the path, ensuring you do not delete existing text or quotation marks.

Example:

“`

“C:Program FilesSteamsteamappscommonGameNameGameExecutable.exe” –windowed

“`

5. Click Apply and then OK.

6. Launch the game via this shortcut; it should now open in windowed mode.

This method provides a quick way to set windowed mode for specific games without altering Steam’s global settings.
